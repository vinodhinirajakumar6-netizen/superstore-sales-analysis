# superstore-sales-analysis
Full-stack data analytics project analyzing Superstore sales using Python for data cleaning, PostgreSQL for querying, and Power BI for interactive dashboard visualization.


#  Superstore Sales & Delivery Analytics Dashboard

##  Project Overview

This project presents an end-to-end data analytics solution for analyzing retail sales and delivery performance. The dataset was cleaned and transformed using Python, analyzed using PostgreSQL, and visualized through an interactive dashboard in Power BI.

---

##  Objectives

* Analyze overall sales performance
* Identify top-performing regions and categories
* Evaluate delivery efficiency
* Provide actionable business insights

---

##  Tools & Technologies

* **Python (Pandas)** → Data cleaning & preprocessing
* **PostgreSQL** → Data analysis & querying
* **Power BI** → Dashboard & visualization

---

##  Data Cleaning & Preparation

* Handled missing values (Postal Code using City & State logic)
* Fixed inconsistent date formats
* Removed duplicates
* Standardized categorical data
* Created new feature: **Delivery Days**

---

##  Key Metrics (KPIs)

*  Total Sales
*  Total Orders
*  Average Delivery Days
*  Sales per Order

---

##  Dashboard Features

* Sales by Category
* Sales by Region
* Monthly Sales Trend
* Delivery Performance Analysis
* Top 10 Cities by Sales
* Interactive Filters (Category, Region, Segment)

---

##  Key Insights

* West region generates the highest revenue
* Technology category contributes the most sales
* Delivery performance improves over time
* Peak periods experience delivery delays

---

##  Project Files

* `clean_superstore.csv` → Cleaned dataset
* `superstore_dashboard.pbix` → Power BI dashboard
* `analysis_queries.sql` → SQL queries used for analysis

---

##  How to Use

1. Load dataset into PostgreSQL
2. Run SQL queries for analysis
3. Open `.pbix` file in Power BI
4. Interact with dashboard using filters
